BuzzFeed: React With a GIF

Role: Creative Director, Creative Lead, Lead Animator, Designer

Working closely with BuzzFeed's Tech and Ad Product teams, I helped develop a brand-new way for audiences to react to content using the Internet's favorite format: animated GIFs. React With a GIF allows for direct brand integration across nearly all content on BuzzFeed, and positions brand assets in a way that encourages the BuzzFeed community to actively engage with them. It's like a trailer, but a lot more fun.
