BuzzFeed: React With a GIF

Role: Creative Director, Creative Lead, Lead Animator, Designer

Working closely with BuzzFeed's exceptional Tech and Ad Product teams, I co-developed a brand-new way to react to content using the Internet's favorite format: the animated GIF.

React With a GIF allows for direct brand integration across nearly all content on BuzzFeed, positioning brand assets as reaction buttons that the BuzzFeed community actively engages with. It's like a trailer — but a lot more fun.
